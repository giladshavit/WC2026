### Prerequisites
- Node.js
- Expo CLI: `npm install -g expo-cli`

### Installation

```bash
cd mobile-app
npm install
```

### Running the App

```bash
# Start Expo development server
npm start

# Run on iOS simulator
npm run ios

# Run on Android emulator
npm run android

# Run in web browser
npm run web
```
